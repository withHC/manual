홈 화면

비디오 수정

테마 디자인 > Theme Editor으로 이동 후 테마 파일 영역에서 assets > js > home.js를 클릭하면 위 화면이 나옵니다.

const id="수정할 부분"

수정할 부분에 유튜브 비디오의 ID를 넣습니다. 해당 비디오의 url에서 watch?v= 뒤에 적힌 부분이 비디오의 ID입니다.

슬라이더(빌보드) 수정

미디어에서 빌보드에 넣을 이미지를 업로드 후, 이미지를 클릭하면 위 화면이 표시됩니다.

빨간 테두리에 표시된 링크를 복사합니다. 링크에 마우스를 올리고 좌클릭 3번이면 간단하게 선택하실 수 있습니다.

테마 디자인 > Theme Editor으로 이동 후 테마 파일 영역에서 홈페이지를 클릭하면 위 화면이 나옵니다.

빨간 테두리 내부를 수정하시면 됩니다.

background-image:url(복사한 url을 여기에 붙여 넣습니다.)

글 추가

뉴스

뉴스에 포함된 유튜브는 디바이스 크기에 맞게 알아서 너비를 맞춰 중앙 정렬됩니다.

별도로 중앙 정렬을 사용하면 레이아웃이 망가지니, 중앙 정렬을 삼가주세요.

아티스트

글 > 새로 추가 후에 우측 화면 문서 > 카테고리에서 해당 아티스트의 분류를 맞춰주세요. (Actor, Musician)

제일 먼저 왼쪽에 보이는 + 버튼 클릭 후 레이아웃 요소 > 미디어 및 텍스트를 클릭합니다.

위 버튼을 클릭해 헤딩 요소로 변경을 합니다.

h3을 클릭 후 아티스트 분류를 적습니다 (Musician, Actor)

엔터 후에 다시 헤딩 요소로 변경 후 h2를 클릭하고, 아티스트의 이름을 적습니다.

만약 멤버가 있다면 위 과정을 반복하되, h3으로 변경하는 과정은 생략합니다.

영문 / 한글 이름을 병기하는 경우, 둘 다 헤딩요소에서 h2를 클릭하여 진행합니다.


그 아래의 내용들은 별다른 수정 없이 입력하시면 됩니다.

소셜 버튼을 삽입할 땐, ctrl + shift + alt + M 을 입력하면 위와 같은 화면이 보입니다.

Profile 영역은 레이아웃 요소 > 열을 사용합니다.

왼쪽엔 Profile, 오른쪽엔 내용을 작성하시면 됩니다.
